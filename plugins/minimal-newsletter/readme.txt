MININMAL NEWSLETTER PLUGIN

STEP 01
================
Newsletter Block

-------------------------------------------------------------------------------------------------------------
Insert a newsletter block to the your page. 
Name the form so you know which signup block is performing best. 
We recommend leaving the title, description and disclaimer fields blank. 
Choose a button label like “submit”, “signup”, or a simple arrow.
-------------------------------------------------------------------------------------------------------------

STEP 02
================
Code Injection

-------------------------------------------------------------------------------------------------------------
Go into Code Injection > Header section and paste this link here:

<link rel="stylesheet" href="https://hulo.dev/plugins/minimal-newsletter/minimal-newsletter.css">
-------------------------------------------------------------------------------------------------------------
